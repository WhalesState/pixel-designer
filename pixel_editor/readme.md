## Pixel Editor Roadmap
- The Editor is a versatile window that serves as both a preview for designers and a platform for expanding functionality. It features a toggle button to easily switch between editing and non-editing modes, allowing the editing tools to be hidden when not needed. To optimize the UI for pixel style, I have modified and created multiple UI classes, reducing their size and ensuring their suitability. By integrating the editor within the main preview, which shares the same UI containers, so we can avoid duplicating the UI and their scripts.
